Artifact 4 — Data Challenge Scenarios: Data Cleaning in Machine Learning

Title

Data Challenge Scenarios – Handling Missing Data, Noisy Sensor Readings, and Outliers

Introduction

This artifact is based on the Workshop 4 Data Challenge Scenarios activity. It demonstrates my ability to analyze real-world data problems involving missing values, noisy measurements, and outliers while selecting appropriate cleaning methods to support reliable machine learning.

Description

I explored three realistic data issues:

  • Missing Data: choosing appropriate imputation techniques and deciding when to drop incomplete rows.
  • Noisy Data: detecting sensor spikes and applying smoothing to keep meaningful trends intact.
  • Outliers: evaluating whether extreme values were valid or errors and assessing impact on analysis.
Objective

Show practical reasoning in data preprocessing—an essential skill for dependable ML workflows.

Process

I analyzed each scenario, compared alternative preprocessing techniques, selected the most appropriate approach, and rewrote the explanations clearly for both technical and non-technical audiences.

Tools & Techniques Used

Imputation, smoothing, outlier analysis, data visualization reasoning, and responsible ML decision-making.

Value Proposition

This artifact shows my ability to clean imperfect data thoughtfully instead of relying on automated tools blindly—an essential foundation for trustworthy ML models.

Unique Value

This piece highlights my decision-making skills and ability to explain why I choose one technique over another in specific real-world scenarios.

Relevance

Supports course outcomes for data preprocessing, ML reasoning, and responsible AI practice. Complements my other artifacts well.
